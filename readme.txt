This payment gateway is used as offline credit card payment for WooCommerce. 

== Description ==

This payment gateway for WooCommerce provide facility to  your customers to enter their credit card information at your site and check data manually as offline using some predefined algorithm. For security reason after use of credit card information you have to delete that notes from the order information of the admin.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/woocommerce offline credit card payment` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->woocommerce offline credit card payment screen to configure the plugin
4. Go to the woocommerce->settings->checkout and click on offline credit card tab.


== Frequently Asked Questions ==

= 1. Where we get credit card information of customer after completion of customer order? =

Go to woocommerce->orders then click on view of button for specific order. You can see the detail of credit card information at order notes section.
